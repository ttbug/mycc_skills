# 小红书常见爆款文案套路

## 一、标题套路库

### 1. 数字式
- "N个...方法/技巧/雷区/秘密"
- "用...天/月/年，我..."
- "...的N种打开方式"
- 例："3个方法让我月入过万"、"30天逆袭指南"

### 2. 疑问式
- "为什么...总是...?"
- "...真的有用吗？"
- "你还在...吗？"
- 例："为什么你护肤没效果？"、"熬夜真的会变丑吗？"

### 3. 对比式
- "别再...了，试试..."
- "...vs...，哪个更好？"
- "以前...现在..."
- 例："别再跟风了，找到自己的风格"

### 4. 情绪式
- "终于...了！"
- "后悔没早知道..."
- "没想到..."
- 例："终于找到适合黄黑皮的口红了！"

### 5. 干货式
- "保姆级教程：..."
- "手把手教你..."
- "...攻略（附清单）"
- 例："保姆级教程：从零开始学化妆"

## 二、开头钩子套路

### 1. 痛点共鸣型
```
"你是不是也...？"
"跟我一样...的姐妹看过来"
"终于找到...了"
```

### 2. 反差惊讶型
```
"没想到...竟然..."
"原来...一直做错了"
"发现了一个...秘密"
```

### 3. 利益承诺型
```
"教你一招..."
"分享一个...方法"
"这个...帮我..."
```

### 4. 真实经历型
```
"作为一名...我..."
"亲测...真的..."
"用...之后，我..."
```

## 三、正文结构模板

### 模板1：问题-解决型
```
开头：提出痛点/问题
中间：
  1. 分析原因
  2. 给出解决方案（分点）
  3. 展示效果/证明
结尾：总结+行动号召
```

### 模板2：体验分享型
```
开头：引入话题/制造期待
中间：
  1. 产品/服务介绍
  2. 使用过程/体验细节
  3. 效果展示（对比）
  4. 优缺点分析（增加可信度）
结尾：推荐+购买建议
```

### 模板3：干货教程型
```
开头：承诺价值/建立信任
中间：
  1. 准备工作
  2. 步骤拆解（图文/分点）
  3. 注意事项/避坑指南
  4. 进阶技巧
结尾：鼓励+互动引导
```

### 模板4：观点输出型
```
开头：抛出观点/引发思考
中间：
  1. 现象描述
  2. 深度分析
  3. 论证支撑（个人经历/数据/案例）
  4. 可行性建议
结尾：价值升华+互动提问
```

## 五、互动引导套路

### 收藏引导
- "干货满满，建议收藏备用"
- "下次用的时候找不到啦"
- "点收藏，随时都能看"

### 点赞引导
- "有用的话点个赞吧"
- "让我知道有没有人看"
- "点赞过xx继续更"

### 评论引导
- "你有哪些...经验？评论区聊聊"
- "你还想了解...？评论区告诉我"
- "第...个你中招了吗？"

### 关注引导
- "关注我，每天分享..."
- "带你了解真实的..."
- "不定期更新..."

## 六、热门领域爆款规律

### 美妆护肤
- **标题**：肤色/肤质+效果承诺
- **内容**：使用前后对比+详细教程+肤质说明
- **关键词**：黄黑皮、干皮、油皮、伪素颜、纯欲

### 时尚穿搭
- **标题**：身材/场景/风格+搭配建议
- **内容**：OOTD+单品链接+搭配技巧
- **关键词**：小个子、梨形身材、通勤、约会、复古

### 美食制作
- **标题**：食材+难度+效果描述
- **内容**：食材清单+步骤图+小贴士
- **关键词**：零失败、保姆级、巨好吃、减脂

### 职场成长
- **标题**：痛点/场景+解决方案
- **内容**：个人经历+方法论+可操作建议
- **关键词**：升职加薪、副业、面试、沟通技巧

### 生活家居
- **标题**：空间/问题+改造效果
- **内容**：改造前后+好物推荐+性价比分析
- **关键词**：租房、收纳、改造、好物

### 情感心理
- **标题**：关系/情绪+共鸣话题
- **内容**：故事+分析+建议
- **关键词**：恋爱、婚姻、独居、焦虑、治愈
