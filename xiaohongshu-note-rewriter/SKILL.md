# 小红书笔记重写器

## 概述

将小红书笔记重写为爆款吸睛风格，提升笔记曝光率和互动潜力。支持多种输入方式，输出包含优化的标题、正文、标签及封面图完整设计方案。

## 工作流程

### 1. 确定输入方式

根据用户提供的信息选择获取内容的方式：

| 用户输入                | 处理方式                                       |
| ----------------------- | ---------------------------------------------- |
| 小红书链接 (http/https) | 使用 `mcp__web_reader__webReader` 工具获取内容 |
| 粘贴的笔记文本          | 直接使用粘贴内容                               |
| 文件路径                | 使用 Read 工具读取文件                         |

### 2. 提取笔记原始内容

从获取的内容中提取：

- 原始标题
- 正文内容（去除HTML标签后）
- 现有标签（如有）

### 3. 重写标题

爆款标题原则：

**标题公式库（混合使用）：**

- 数字冲击型：`"X个___，我后悔知道得太晚"` / `"用完___，我的___发生了巨变"`
- 痛点共鸣型：`"终于找到___的解决办法了"` / `"___人群集合！这个真的有用"`
- 反差对比型：`"本来以为___，结果___"` / `"千万别___，除非___"`
- 紧迫感型：`"再不看就晚了！___"` / `"手慢无！___"`
- 好奇悬念型：`"没人告诉我___原来这么___"` / `"发现一个___的秘密"`

**标题要求：**

- 15-20字最佳,包含符号和emoji
- 包含关键词（SEO）
- 情绪化表达（激动/惊讶/紧迫）
- 适当使用 emoji（1-2个）

### 4. 重写正文内容

**结构框架：**

```
开头钩子（2-3行）
- 制造共鸣/痛点/好奇
- 与封面图呼应

核心内容（分点阐述）
✅ 要点1 + emoji
   - 增加适用场景, 让人能够对号入座，觉得感同身受
✅ 要点2 + emoji
   - 增加适用场景，让人能够对号入座，觉得感同身受
✅ 要点3 + emoji
   - 增加适用场景，让人能够对号入座，觉得感同身受


评论钩子（2-3行）
- 提供选择/制造共鸣/问题求助
```

**写作原则：**

- 段落简短，每段不超过3行
- 大量使用分段符和空行
- emoji 适度但频繁使用（每段1-2个）
- 口语化表达，像朋友聊天
- 重点内容用emoji或特殊符号标注

**实用技巧：**

- 用"我真的"增加真实感
- 用"绝了" "太香了" "yyds"等口语化表达
- 用 | 或 // 或 → 等符号增强视觉节奏
- 数字和关键信息用emoji突出

### 5. 优化标签

标签策略：

- **核心标签（3-5个）**：直接相关的高频关键词
- **场景标签（2-3个）**：使用场景、人群定位
- **长尾标签（2-3个）**：细分领域、竞争较小的词

格式：`#标签1 #标签2 #标签3`

### 6. 生成封面图设计建议

输出完整可执行的设计方案，包含：

**设计概述**

- 设计风格定位（如：活力渐变风、极简ins风、复古杂志风）
- 色彩方案（主色/辅色/背景色的色值）
- 目标视觉感受

**元素规格**

- 背景处理（纯色/渐变/图片底图，含具体参数）
- 主标题文字（字体、字号、颜色、位置）
- 副标题/标签文字（样式说明）
- 装饰元素（图标、形状、贴纸的描述和位置）
- 图片比例3:4，2K高清

**设计要点**

- 对齐方式
- 留白处理
- 对比度检查
- 可读性保证

## 输出格式

```markdown
---
## 📝 重写后的小红书笔记

### 标题
[优化后的标题]

### 正文
[重写后的正文内容]

### 标签
[优化后的标签]

---

## 🎨 封面图设计建议

### 设计概述

[设计风格和定位描述]

### 色彩方案

- 主色：#XXXXXX
- 辅色：#XXXXXX
- 背景：#XXXXXX

### 元素规格

1. **背景**：[描述 + 参数]
2. **主标题**：[字体、大小、颜色、位置]
3. **装饰**：[描述 + 位置]

### 设计要点

- [要点1]
- [要点2]

---
```

## 注意事项

- 保持原意核心信息，不编造不存在的内容
- 避免过度夸张导致虚假宣传
- 标签要与内容真实相关
- 封面设计建议要具体可执行，避免模糊描述
- 笔记内容不要使用小红书高风险词语,参考`references/sensitive-words.md`
