## 检测策略

1. **行为模式识别**:
   - 视频结尾固定引流话术
   - 评论区置顶引流信息
   - 主页/简介引流检测

2. **画面检测**:
   - 二维码识别
   - 联系方式识别
   - 竞品logo识别
   - 水印检测

3. **语音检测**:
   - 口播引流话术
   - 口播联系方式

4. **合规建议**:

| 违规行为 | 合规替代 |
|---------|---------|
| 加微信 | 使用抖音私信/粉丝群 |
| 评论扣1 | 欢迎在评论区交流 |
| 点赞关注 | 感谢大家的支持 |
| 外部链接 | 使用小黄车/橱窗功能 |
| 未标广告 | 视频标注"广告/推广" |
