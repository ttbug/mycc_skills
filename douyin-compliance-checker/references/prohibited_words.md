# 违禁词库

本词库包含涉政、涉恐、涉黄、暴力血腥等敏感内容关键词。

## 使用说明

- **风险等级**: HIGH(高危-直接违规), MEDIUM(中危-可能违规), LOW(低危-建议避免)
- 匹配时建议使用模糊匹配，考虑变体写法（如用谐音、拼音、emoji替代）

---

## 涉政敏感词 (HIGH)

> 注意：此类内容直接违规，会导致账号封禁

```
国家领导人姓名变体
敏感历史事件
分裂国家相关
颠覆政权相关
境外敏感组织
```

**检测建议**: 对于涉政内容，任何疑似提及都应标记为高风险。


## 检测策略

1. **精确匹配**: 直接命中词库中的词汇
2. **变体检测**: 检查谐音、拼音、emoji替代
3. **上下文分析**: 结合上下文判断是否真正违规（如新闻报道vs宣扬）
4. **图片OCR**: 对视频画面中的文字也进行检测
