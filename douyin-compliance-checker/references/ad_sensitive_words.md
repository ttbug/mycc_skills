# 广告法敏感词库

本词库基于《中华人民共和国广告法》及抖音平台广告规范整理。

## 使用说明

- **风险等级**: HIGH(直接违规), MEDIUM(可能违规), LOW(建议避免)
- 适用于：商品推广、带货视频、品牌宣传等商业内容

---

## 绝对化用语 (HIGH)

> 《广告法》第九条明确禁止

### 极限词
```
最、最佳、最好、最优、最优秀、最高、最高级、最高端
最低、最低价、最低级、最便宜
最大、最小、最强、最先进
第一、No.1、TOP1、排名第一
唯一、独一无二、独家、独创、首创、首选、首发
绝对、绝佳、绝无仅有
极、极品、极致、极佳、极度、极端
顶、顶级、顶尖、顶端、巅峰
冠、冠军、领先、领袖、之王、王者
```

### 时间极限词
```
空前、绝后、永久、永恒、万能、无限
史无前例、前无古人、万人空巷
```

## 虚假承诺 (HIGH)

### 效果承诺
```
100%、百分百、100%有效
保证、承诺、绝对有效
无效退款、假一赔十（无法兑现时）
立竿见影、立刻见效、一次见效
永不反弹、根治、治愈、药到病除
```

### 收益承诺
```
稳赚、躺赚、保本、保收益
日赚XXX、月入XXX（夸大收益）
零风险、无风险投资
```

## 医疗健康类违禁词 (HIGH)

> 非医疗机构/药品不得使用

### 功效词
```
治疗、治愈、疗效、药效
预防疾病、抗癌、防癌
消炎、杀菌、抑菌（非消毒产品）
降血压、降血糖、降血脂
减肥、瘦身、燃脂（需资质）
```

### 医疗术语
```
处方、临床、医学证明
专家推荐、医生建议（无真实背书）
```

```
速效、超强、全效、特效、神效
换肤、抗皱、除皱、去皱
美白（需特证）、祛斑（需特证）
再生、重生、新生
基因、干细胞、活细胞
```

```
纯天然、100%天然
零添加、无添加（绝对化）
医学级、医美级（非医疗器械）
```

## 检测策略

1. **精确匹配**: 检测上述词汇的直接使用
2. **语境判断**:
   - "这是我用过最好的" → 违规
   - "这是我个人最喜欢的" → 可能通过
3. **图片文字**: OCR检测商品包装、海报上的违禁词
4. **口播检测**: 语音转文字后检测
5. **修改建议**: 提供合规替代表达
